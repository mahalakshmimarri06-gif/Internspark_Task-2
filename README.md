Crypto Market Tracker

This project is a Python-based CLI application that fetches live cryptocurrency market data from the CoinGecko API. 
It allows users to view top market performers and filter results by specific cryptocurrency names.

FeaturesFetches real-time crypto market data using the requests library.  

Parses JSON responses to extract coin names and current prices.
Provides a search/filter functionality for specific cryptocurrencies.  
Professional console-based output formatting. 
PrerequisitesEnsure you have Python installed. 
You will need the requests library:
pip install requests
